PART 4: JAVASCRIPT MODULE STRUCTURE (know3_enhancements.js)
================================================================================

MODULE EXPORTS:

window.enhancedSearch — Main API object
  .enableBackend(url) → Promise<bool>
  .ingestFileEnhanced(file) → Promise<{success, chunks?, error?}>
  .hybridRAGSearch(message, topK, collection) → Promise<{success, results?, error?}>
  .getStats() → {bm25Collections, backendActive, weights, ...}

window.BM25SearchEngine — BM25 search engine
  .addDocument(collection, text, metadata)
  .collections — Internal collections object
  .tokenize(text) → [tokens]

window.HybridSearchEngine — Hybrid search engine
  .hybridSearch(query, embedding, topK, collection) → Promise<[results]>
  .bm25Weight — Configurable weight (default 0.6)
  .vectorWeight — Configurable weight (default 0.4)
  .rerankerK — Configurable reranking depth (default 10)

HYBRID SEARCH SCORING ALGORITHM:

Step 1: Tokenize Query
  tokens = tokenize(query)  // Remove stopwords, split

Step 2: BM25 Scoring
  For each document:
    score_bm25 = SUM[ IDF(term) * (freq*(k1+1)) / (freq + k1*(1-b+b*|D|/avgLen)) ]
  Where:
    k1 = 1.5 (term saturation)
    b = 0.75 (length normalization)
    IDF(term) = ln((N - df + 0.5) / (df + 0.5))

Step 3: Vector Scoring (if embedding available)
  score_vector = cosine_similarity(query_embedding, doc_embedding)

Step 4: Normalize & Combine
  score_bm25_norm = min(1.0, score_bm25 / 10.0)
  score_vector_norm = score_vector  (already in [0, 1])
  score_combined = (0.6 * score_bm25_norm) + (0.4 * score_vector_norm)

Step 5: Rerank for Diversity
  For result in top-10:
    If Jaccard(result.text, selected_text) > 0.85:
      Skip (too similar)
    Else:
      Include in final top-k
